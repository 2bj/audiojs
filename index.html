audio.js

audiojs is a drop-in javascript library that allows HTML5's <audio> tag to be used anywhere.

Usage

<!DOCTYPE html>
<head>
  <meta charset="utf-8">
  <title>audio.js</title>
  <script src="/audiojs/audio.js"></script>
  <script>
    audiojs.events.ready(function() {
      audiojs.createAll();
    });
  </script>
</head>
<body>
  <audio src="/mp3s/juicy.mp3" preload="auto" />
</body>
